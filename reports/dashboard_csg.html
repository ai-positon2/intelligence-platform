CSG Dashboard

This account is ready to be set up.
Follow the steps below to generate the CSG intelligence dashboard.

Setup Steps
1
Add your company CSV
Place your Apollo export for CSG companies at:
apollo-accounts-export-csg.csv
2
Configure the account
Update config.yaml with a csg: section pointing to the new CSV and its Google Sheets IDs.
3
Build the dashboard
Run the tracker for CSG:
python main.py --account csg --dashboard-only
4
Deploy
Push to Railway and the CSG dashboard will go live instantly:
git add . && git commit -m "Add CSG dashboard" && git push
